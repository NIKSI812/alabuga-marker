# Alabuga War Bloggers Marker

Расширение для Chrome, которое помечает 💩 YouTube-каналы, 
рекламировавшие ОЭЗ «Алабуга» после осени 2023 года.

## Установка

1. Скачай ZIP
2. Распакуй в любую папку
3. Открой в Chrome: `chrome://extensions/`
4. Включи **«Режим разработчика»** (тумблер справа сверху)
5. Нажми **«Загрузить распакованное расширение»**
6. Выбери распакованную папку

Зайди на YouTube — у каналов из списка появится плашка 💩.

## Поддержка
- Chrome / Edge / Opera / Brave (любой Chromium-браузер)
